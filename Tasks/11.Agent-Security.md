## Task 11: security scanning integration

Mission:
Integrate ClawSec security scanning into the OpenClaw and Hermes agent stacks under `~/projects/noesis-ansible/`. NoesisPraxis is the Hermes supervisor agent for the Noesis Universe infrastructure stack, and relevant agent workflows must be scanned and governed using ClawSec before deployment or promotion.

Canonical references:
- ClawSec: https://prompt.security/clawsec
- ClawSec docs: https://github.com/prompt-security/clawsec/wiki

Requirements:
1. Add Ansible roles, tasks, templates, or validation hooks to enable ClawSec for OpenClaw and Hermes workflows.
2. Ensure security scanning is available wherever agent prompts, skills, MCP toolchains, or runtime configuration change.
3. Keep the implementation idempotent, auditable, and safe to rerun.
4. Make validation explicit so ClawSec coverage can be checked during maintenance or CI-style runs.

Acceptance criteria:
- OpenClaw and Hermes security scanning paths are represented in code.
- ClawSec is integrated into the repo’s operational flow.
- Validation confirms the scanning service is present and correctly wired.
