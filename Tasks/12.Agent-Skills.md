## Task 12: dynamic skill discovery and integration

Mission:
Integrate dynamic skill search, evaluation, and installation into the OpenClaw and Hermes stack using SkillNet and SkillNet MCP. NoesisPraxis is the Hermes supervisor agent for the Noesis Universe infrastructure stack, and the repo must be able to discover, evaluate, and integrate skills as part of agent operations.

Canonical references:
- SkillNet: https://github.com/zjunlp/SkillNet
- SkillNet MCP: https://github.com/CycleChain/skillnet-mcp

Requirements:
1. Add Ansible-managed scaffolding for SkillNet and SkillNet MCP integration.
2. Support skill search, skill installation, and skill evaluation workflows for agents.
3. Represent any required config, environment variables, or service definitions in the repo.
4. Keep skill discovery and integration available as a reusable operational capability for OpenClaw and Hermes.
5. Ensure the result is idempotent, auditable, and safe to rerun.

Acceptance criteria:
- Dynamic skill search is available to the agent stack.
- SkillNet MCP is represented and deployable.
- Skill discovery/integration is encoded in Ansible-managed infrastructure.
- Validation can confirm the skill service is reachable and usable.
