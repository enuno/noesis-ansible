# NoesisLab Brand Kit

This package contains a vector interpretation of the NoesisLab logo and a social media kit generated from the current NoesisLab raster logo. The assets preserve the NoesisPraxis-inspired identity: **deep navy beveled app-icon geometry**, **warm gold/copper metallic emblem work**, a **Greek-key cognition motif**, a **gear/claw engineering arc**, and an **eye-brain-lab symbol** adapted for the NoesisLab local OpenClaw agent.

## Recommended usage

Use `NoesisLab-logo-vector.svg` wherever SVG is accepted, especially documentation, landing pages, and internal app surfaces. Use `NoesisLab-mark-vector.svg` for compact placements where the wordmark becomes too small. For social profiles, use the 800 px or 1024 px profile icon and let the platform downsample it.

The vector logo is a **clean geometric vector recreation**, not an automatic photorealistic trace of the generated PNG. This makes it more editable and scalable, while the social kit uses the generated logo for maximum continuity with the original 3D NoesisPraxis-style aesthetic.
